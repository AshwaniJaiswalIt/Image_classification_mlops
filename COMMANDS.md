# Quick Start Commands

## Setup Environment
```bash
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run Jupyter Notebook (Tasks 1-4)
```bash
jupyter notebook assignment1.ipynb
# Run all cells
```

## Run MLflow UI
```bash
mlflow ui
# Open: http://localhost:5000
```

## View Results
- Models: `*.pkl` files
- Config: `preprocessing_config.json`
